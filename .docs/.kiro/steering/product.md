# Product Overview

This is a PDF processing application that creates a semantic vector database for PDF documents. The system extracts text from PDF files, processes it into chunks, generates embeddings using OpenAI's API, and stores the results in MongoDB for semantic search capabilities.

## Key Features
- PDF text extraction and processing
- Text chunking with configurable overlap
- OpenAI embeddings generation
- MongoDB storage with vector embeddings
- Processing logging and tracking
- Support for both local files and URLs (planned)

## Target Use Case
The application is designed for building a local semantic search system over PDF document collections, particularly useful for research, documentation analysis, and knowledge management.
